# Aspose.HTML for Java – Message Handlers

This gist repository contains practical Java snippets that show how to process web service requests and responses using various message handlers, including logging execution time, implementing custom schemes, working with ZIP archives, managing network operation timeouts, and handling password-based authentication schemes using [Aspose.HTML for Java](https://products.aspose.com/html/java/).

## What's Included?

In this repository, you will find Java code illustrating various popular uses of message handlers in your applications. The examples cover a wide range of functions, including:

* **Custom Schemes**: Create a custom message handler to process requests for a new URI scheme.
* **ZIP Archive Handling**: Efficiently work with HTML documents and their resources (images, styles, scripts) that are packed within a ZIP archive. This includes examples for converting HTML from a ZIP archive to PDF and JPG.
* **Authentication**: Implement a simple message handler for web requests that use password-based authentication schemes.
* **Performance Monitoring**: Track and log the total execution time of a web request to monitor performance.
* **Network Timeouts**: Set and manage network timeouts to ensure a stable and reliable connection.
* **Network Request Inspection**: Intercept and inspect network requests, allowing you to block unwanted external requests.

Aspose.HTML for Java is a powerful, on-premise Java library for parsing, navigating, processing, and converting HTML documents.
