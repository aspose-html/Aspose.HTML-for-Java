# Aspose.HTML for Java – Getting Started

This gist repository contains ready-to-use Java snippets to help you get started with Aspose.HTML for Java: apply a license, use metered licensing, and convert HTML to PDF. These examples are used in the [Getting Started](https://docs.aspose.com/html/java/getting-started/) chapter of the documentation and are intended for quick experimentation and learning.

## Topics Covered

1. Apply license files from disk or stream.
1. Use metered licensing with public/private keys.
1. Convert HTML to PDF and other formats.

## Prerequisites

* J2SE 1.8 or higher.
* Supported OS: Windows, macOS, Linux.
* Java IDE (IntelliJ IDEA, Eclipse, or similar).
* Build tool: Maven or Gradle for dependency management.

## How to Use These Examples

1. Install or reference [Aspose.HTML for Java](https://releases.aspose.com/html/java/) in your project – download packages and view release notes on the product releases page.
2. Configure the Aspose repository and add the Aspose.HTML dependency as described in the official [installation guide](https://docs.aspose.com/html/java/getting-started/installation/).
3. Browse the available gists and copy the code samples you need.
4. Open the example files in your IDE, update file paths or output locations, and run the samples.

## Licensing and Evaluation

You can evaluate Aspose.HTML for Java with a [temporary license](https://purchase.aspose.com/temporary-license/) (30-day full-feature trial). For production use, apply a purchased license or use metered licensing as shown in the examples.
